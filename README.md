# Password-Generator
This is  a password generator created using Python

## Python Libraries Used
- Tkinter
- Random

## How to use (without GUI)
1. Start the program by double clicking `Password Generator.py`
2. Wait...
3. Enter the length of the password that you would like
4. Wait...
5. Enter the numbers of the password that you would like
6. Wait...
7. Here's your password.

## How to use (with GUI)
1. Start the program by double clicking `Password Generator GUI.py`
2. Wait...
3. Enter the length of the password that you would like
4. Press `Get Password`
5. The password appeared down at the entry column.
6. For more password generated, click a few more times.
